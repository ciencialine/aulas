# Contribuindo

## Requisitos

* Ter conta no GitHub.
* NotebookLM público, focado em um conteúdo de aula referente a uma habilidade BNCC.

## NotebookLM

Com uma conta google, é possível criar um notebook LM gratuitamente, com algumas limitações diárias.

A versão paga do pacote google de inteligência artificial libera mais recursos por dia.

[https://notebooklm.google.com/](https://notebooklm.google.com/)
